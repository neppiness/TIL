# **Solution for boj.kr/5648**

### My solution
- Input causes overflow when it is put to `int` type variable
  * Thus, `long long` type should be used
- My source code is written as below:
```cpp
#include <iostream>
#include <string>
#include <vector>
#include <algorithm>

using namespace std;

int main(){
    ios::sync_with_stdio(0); cin.tie(0);

    int N; cin >> N;

    vector<long long> V;

    while(N--){
        string str; cin >> str;

        int m = str.length() - 1;
        long long p10 = 1;

        long long num = 0;
        for(auto it = str.begin(); it < str.end(); it++) {
            num += (*it - '0') * p10;
            p10 *= 10;
        }
        V.push_back(num);
    }

    sort(V.begin(), V.end());
    for(auto a : V) cout << a << '\n';
}
```

### Solution source code
```cpp
// Authored by : std-freejia
// Co-authored by : -
// http://boj.kr/c56909c54c4f4e9fb2c987a6eb3b96ee
#include <bits/stdc++.h>
using namespace std;

int n;
string st;
vector<long long> v;

int main(void) {
  ios::sync_with_stdio(0);
  cin.tie(0);
  cin >> n;
  for(int i = 0; i < n; i++) {
    cin >> st;
    reverse(st.begin(), st.end());
    v.push_back(stoll(st));
  }
  sort(v.begin(), v.end());
  for(auto i : v) cout << i << '\n';
  return 0;
}
```
