  - insert 함수에서 가장 중요한 `cur = nxt;`를 안 넣었었음. 구현 실수 주의바람.
```cpp
void insert(string &s, bool code) {
  int cur = ROOT;
  for(char c : s) {
    int nxt = find(c, cur);
    if(nxt == -1) { // -1: not_found
      nxt = unused;
      trie[cur].push_back({c, unused++});
    }
    chk[nxt] = code;
    cur = nxt;
  }
}
```
